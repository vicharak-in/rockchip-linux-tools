# Rockchip Developer Tools

## Introduction

This repository contains the executable binaries for Rockchip Developer Tools.
These tools are used for different purposes, such as:

- Flashing firmware to Rockchip devices
- Formatting partitions
- Creating images for Rockchip devices
- Debugging Rockchip devices
- Sign/Encrypt images for Rockchip devices
  and more.
